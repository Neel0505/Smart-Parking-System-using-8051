# 🚗 Smart Parking System using 8051 (Proteus + Keil)

🚀 This project is created to help students easily build a working 8051 mini project for MPMC / Microcontroller subjects.

It is a **fully working Smart Parking System simulation** using Embedded C, Proteus, and Keil.

---

## 📌 What This Project Does

* Displays available parking slots on LCD
* Entry button → decreases count
* Exit button → increases count
* Motor simulates gate opening/closing
* Real-time updates on 16x2 LCD

---

## 🎯 Perfect For

* MPMC Mini Projects
* Viva preparation
* Beginners learning 8051
* Students stuck with Proteus + Keil setup

---

## 🛠️ Technologies Used

* 8051 Microcontroller (AT89C51)
* Embedded C
* Proteus Design Suite
* Keil uVision
* L293D Motor Driver
* 16x2 LCD

## ⚡ How to Run (Step-by-Step)

1. Open Proteus project file
2. Open code in Keil
3. Build the project (F7) → generate HEX file
4. Load HEX file into 8051 in Proteus
5. Click RUN ▶

---

## 🎮 How to Use

* Press ENTRY button → slots decrease
* Press EXIT button → slots increase
* Motor rotates to simulate gate

---

## ⚠️ Common Mistakes (IMPORTANT)

* ❌ EA pin not connected to +5V
* ❌ LCD V0 not connected to GND
* ❌ Buttons without pull-up resistors
* ❌ Wrong HEX file loaded

👉 Fix these if your project doesn’t run
